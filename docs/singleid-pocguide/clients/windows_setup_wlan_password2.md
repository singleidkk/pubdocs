# Windows: パスワードで無線LANアクセスの認証（詳細設定）

以下、Windowsに無線LANの802.1X認証設定を手動で追加する方法です。

1. Windowsの**スタートメニュー＞アプリ、設定、ドキュメントの検索**より、**control**を入力し、Enterキーを押下します。
2. **コントロール パネル**が表示されます。**ネットワークとインターネット＞ネットワークと共有センター**をクリックします。
3. **新しい接続またはネットワークのセットアップ**をクリックします。
4. **ワイヤレス ネットワークに手動で接続します**を選択し、**次へ**ボタンをクリックします。
5. 以下情報を設定し、**次へ**ボタンをクリックします。

    | **設定項目** | **設定内容** |
    | :--- | :--- |
    | **ネットワーク名** | 接続する無線LANアクセスポイントのSSIDを入力します。 |
    | **セキュリティの種類** | 無線LANアクセスポイント側の設定に合わせます。通常は**WPA2-エンタープライズ**を選択します。 |
    | **暗号化の種類** | 無線LANアクセスポイント側の設定に合わせます。通常は**AES**を選択します。 |
    | **この接続を自動的に開始します** | 必要に応じてチェックします。 |

6. **接続の設定を変更します**をクリックします。
7. **ワイヤレス ネットワークのプロパティ**画面の**セキュリティ**タブで、以下を設定し、**設定**ボタンをクリックします。

    | **設定項目** | **設定内容** |
    | :--- | :--- |
    | **セキュリティの種類** | 手順5で設定した内容が表示されていることを確認します。 |
    | **暗号化の種類** | 手順5で設定した内容が表示されていることを確認します。 |
    | **ネットワークの認証方法の選択** | **Microsoft: 保護された EAP (PEAP)**を選択します。 |

8. **保護された EAP のプロパティ**画面で、以下情報を設定し、**構成**ボタンをクリックします。

    | **設定項目** | **設定内容** |
    | :--- | :--- |
    | **証明書を検証してサーバーのIDを検証する** | チェックします。 |
    | **次のサーバーに接続する** | チェックし、**SingleID 管理者ポータル＞認証＞RADIUS**画面の**基本情報**タブの**ホスト名**を入力します。 |
    | **信頼されたルート証明機関** | **ISRG Root X1**を選択します。 |
    | **認証方法を選択する** | **セキュリティで保護されたパスワード (EAP-MSCHAP v2)**を選択します。 |
    | **高速再接続を有効にする** | 必要に応じてチェックします。 |

    !!! info
        SingleIDのRADIUSサーバ証明書はLet's Encryptで発行されるため、短い周期で定期的に更新されます。**次のサーバーに接続する**にRADIUSサーバのホスト名を入力することで、証明書更新後も同じホスト名のサーバ証明書として検証できます。

9. **EAP MSCHAPv2 のプロパティ**画面で、**Windows のログオン名とパスワードを自動的に使う**のチェックを外し、**OK**ボタンをクリックします。

    !!! info
        このチェックを外すことで、Windowsのログオン情報ではなく、接続時に入力するSingleIDのユーザ名とパスワードで認証できます。

10. **保護された EAP のプロパティ**画面で、**OK**ボタンをクリックします。
11. **ワイヤレス ネットワークのプロパティ**画面の**セキュリティ**タブに戻ります。**詳細設定**ボタンをクリックします。
12. **802.1Xの設定**タブで、以下情報を設定し、**OK**ボタンをクリックします。

    | **設定項目** | **設定内容** |
    | :--- | :--- |
    | **認証モードを指定する** | チェックします。 |
    | **認証モード** | **ユーザ認証**を選択します。 |

13. **ワイヤレス ネットワークのプロパティ**画面で、**OK**ボタンをクリックします。
14. 無線LANアクセスポイントのSSIDを選択し、接続します。認証情報の入力画面が表示された場合は、SingleIDの**ユーザ名**と**パスワード**を入力します。
15. 接続成功したことを確認します。
