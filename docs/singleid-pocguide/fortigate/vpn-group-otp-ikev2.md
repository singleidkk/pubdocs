# リモートアクセスVPN-2要素認証（パスワード認証＋ワンタイムパスワード認証）（IKEv2自動トランスポートモード）
文書更新日:2026-06-22

## 本手順について

FortiGate のリモートアクセス VPN（IPsec）で、**IKEv2 の自動トランスポートモード**を利用する構成では、SingleID の**2要素認証（パスワード認証＋ワンタイムパスワード認証）**は利用できません。

## 理由

SingleID のワンタイムパスワード認証では、RADIUS 認証方式として **PAP** を利用する必要があります。

一方、FortiGate のリモートアクセス VPN（IPsec）の **IKEv2 自動トランスポートモード**では、この構成で必要となる PAP 認証を利用できません。そのため、**標準 RADIUS サーバ**を利用する構成でも、SingleID のワンタイムパスワード認証は利用できません。
